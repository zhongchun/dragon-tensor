## Overview

The API documentation provides comprehensive reference materials for both C++ and Python interfaces:

- **C++ API**: Core tensor operations, storage backends, buffer management, and I/O operations
- **Python API**: High-level interfaces for tensor creation, operations, interop with NumPy/PyTorch/Arrow, and financial analysis

Both APIs support:
- Zero-copy interoperability with NumPy, PyTorch, and Apache Arrow
- Multiple storage backends (memory, mmap, shared memory, Arrow/Parquet)
- Financial operations (returns, rolling statistics, correlation, covariance)
- Efficient memory management and layout control
